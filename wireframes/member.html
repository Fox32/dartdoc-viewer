dart:io File create()

abstract Future<File> create()

Description
Create the file.

Existing files are left untouched by create. Calling create on an existing file might fail if there are restrictive permissions on the file.
Return Type
Returns a Future<File> that completes with the file when it has been created.
Examples
Example to create and use a file: 

new File("example.txt").create().then((File newFile) {
  OutputStream stream = newFile.openOutputStream();
  
  stream.writeString("text");
  
  stream.close();
});
